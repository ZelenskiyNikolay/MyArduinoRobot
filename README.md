Pet Robot

This project is a small pet robot.

Features:

-Reacts to motion and changes states based on the time of day.

-Produces R2D2-style sounds.

-Changes facial expressions on the screen depending on its current state.

In development:

-Movement on a table, dancing, and other animations.

-Autonomous charging: parking on the charger in the evening or when the battery is low.

-PC communication via radio link for extended functionality.

Modules used:

Arduino Mega 2560

OLED SPI (0.96" and 1.3")

RTC module

Buzzer for sounds

Робот-питомец

Мой проект — небольшой робот-питомец.

Функции:

-Реагирует на движения и меняет состояния в зависимости от времени суток.

-Издает звуки в стиле R2D2.

-Меняет "мордашки" на экране в зависимости от состояния.

В разработке:

-Движение по столу, танцы и другие анимации.

-Самостоятельная зарядка: парковка на зарядку вечером или при разряде аккумулятора.

-Связь с ПК по радиоканалу для расширения функционала.

Используемые модули:

Arduino Mega 2560

OLED SPI (0.96" и 1.3")

Часы RTC

Зуммер для звука
